# AIMarket Frontend Update Pack

## Added
- Soft premium light theme with pastel/white background.
- Region pricing display: BDT for Bangladesh, PKR for Pakistan, Worldwide price for others.
- Checkout payment method is locked by detected region.
- Floating WhatsApp support popups on checkout, pending, and status pages:
  - Left: BD + Worldwide support
  - Right: Pakistan support
- No demo products. Empty state until admin adds products.
- Improved delivery page with large email/password card, copy/show controls, and direct embedded video/image guide.
- Owner-only Admin Users page.
- Product form now supports BDT, PKR, Worldwide price and original prices.
- Smart dashboard with product-wise sold count and revenue.
